Object Teams Development Tooling

The OTDT is a development environment for the OT/J programming language.

Based on the Eclipse Java development tools (JDT), it includes the following key features, which are explained in the next chapters.

Key features of the OTDT

More details can be found on the comprehensive feature list.
  • Creating an Object Teams project using a wizard
  • Creating team and role classes using a wizard
  • Editing OT/J:
    • Syntax highlighting, navigation, content/code assist, base code annotations
    • A table based Binding Editor specifically supports the creation of connectors
    • Comparing bound methods
  • Object Teams structural/navigational views (package explorer, type hierarchy, outline)
  • Compiling Object Teams programs
  • Running Object Teams programs
  • Debugging Object Teams programs
The following feature -- although available in the JDT -- is only partially adapted for the OTDT:
  • Refactoring Object Teams programs

Prerequisites

If you have never used the eclipse Java IDE before, please consider getting familiar with the Java Development Tooling, as the OTDT is built on top of and extends the JDT.

Feedback

If you have questions regarding the OT/J language or if you experience problems or bugs with the OTDT, do not hesitate to send a mail to the otj-users@objectteams.org mailing list.
